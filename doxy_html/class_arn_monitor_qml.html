Detailed Description

ARN Monitor QML.

This class is the Qml version of the ArnMonitor.

See also:
ArnQml

Example usage

// In Qml
//
import QtQuick 2.0
import ArnLib 1.0
Rectangle {
width: 370; height: 400
ArnMonitor {
clientId: "std"
monitorPath: "//Test/List/"
onArnChildFound: console.log("Found list item: " + path);
}
}
