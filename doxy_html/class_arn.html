About Destroy

This singleton class is the main reference to the Active Registry Network. All objects are stored in a tree hierarchy and the naming is similar to typical filesystems e.g. "//Measure/Water/Temperature/value". Each part in the given path is dynamically added as needed.

bool Arn::isProviderPath ( const QString &  path)
static

Test if path is.

A bidirectional Arn Data Object is actually a double object, a twin. Each part has it's own path but there life span is depending on each other. One part is the normal "official" and the other part is provider. The provider has an added "!" to the normal path e.g. "//Measure/Depth!" Data written to one part ends up in the other. When a provider is connected to the provider part, it will receive data to process and it will write the result to the same provider part. This way the result will end up in the normal part. This functionality can be used for data validation and limiting.

Parameters:
[in]path.
Return values:
Trueif path is a providerPath, i.e ends with a "!"

ServerMain.cpp.
